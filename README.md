# flap_mastu_bes
FLAP extension library for dealing with beam emission spectroscopy (BES) and other diagnostics data from MAST/MAST-U experiments.

## Getting started
See `test/test_mastu_bes.py` for getting started, and for the solution of some common problems specific to data from these experiments.

## See also
See also the [documentation for FLAP](https://fusion-flap.github.io/flap/) for general usage.
